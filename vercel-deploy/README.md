# Yarn Dye Studio Manager - Vercel Deployment

## Quick Deploy Instructions

### Method 1: Drag & Drop (Easiest)
1. Go to https://vercel.com/new
2. Drag the entire `vercel-deploy` folder into the upload area
3. Click "Deploy"
4. Done! Your app will be live at: https://your-project-name.vercel.app

### Method 2: GitHub (Recommended for Updates)
1. Create a new repository on GitHub
2. Upload these files to the repository
3. Go to https://vercel.com/new
4. Import your GitHub repository
5. Click "Deploy"
